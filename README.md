# ChatGPTs

ChatGPTs is a QtWebEngine-based project, a collection of GPTs. Currently includes [ChatGPT](chat.openai.com), [Doubao](www.doubao.com), [文心一言](yiyan.baidu.com), [通义千问](tongyi.aliyun.com).
